How to save

  1. Click Export games.json. Your browser downloads the updated file.
  2. In your local repo, replace data/games.json with the downloaded file.
  3. Regenerate the JS mirror: printf 'window.GAMES = ' > data/games.js && cat data/games.json >> data/games.js && printf ';\n' >> data/games.js
  4. git add data/games.json data/games.js && git commit -m "add game" && git push
  5. GitHub Pages rebuilds in ~30 sec. Stats + Games pages show the new game.
