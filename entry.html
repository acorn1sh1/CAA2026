Manual publish (if GitHub not configured)

  1. Click Save Lineup or Save Stats. Your browser downloads games.json.
  2. In your local repo, replace data/games.json with the downloaded file.
  3. Regenerate the JS mirror: printf 'window.GAMES = ' > data/games.js && cat data/games.json >> data/games.js && printf ';\n' >> data/games.js
  4. git add data/games.json data/games.js && git commit -m "update lineup/stats" && git push
  5. GitHub Pages rebuilds in ~30 sec.
